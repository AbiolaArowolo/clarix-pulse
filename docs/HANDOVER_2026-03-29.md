## Recovery if the laptop is lost

GitHub backup status:

- latest code is already pushed to GitHub at commit `c232fc7`

Important limitation:

- GitHub does **not** contain `.env.local` secrets
- keep VPS credentials, SMTP settings, and signing secrets backed up separately in a password manager or secure vault

Redeploy recovery path from a new machine:

1. clone the repo from GitHub
2. recreate `.env.local` with the VPS and app secrets
3. create a release archive from the desired commit
4. run `python scripts\vps_clean_redeploy.py --archive <archive-path>`
